Submit an Issue

You can submit an issue to the issue tracker to report a bug or to request a new feature. Please search through the existing issues before posting a new one. Provide as much detail as possible with your issue. For bug reports, provide details as to what happens and when it happens, and if possible, provide a link to a repository with an example. For feature requests, explain in detail what the feature should accomplish and why it should be implemented.
